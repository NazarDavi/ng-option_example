##Angular-онавтика  

Это простой пример использования ng-options  

Для того, чтобы посмотреть пример, необходим [NodeJS](https://nodejs.org/en/download/)  

Дальше проще:  

1. Склонируйте репозиторий;
2. Установите gulp глобально `npm install gulp -g`;  
3. Выполните в папке с примером `npm install`;
4. Выполните в папке с примером `bower install`;
5. Запустите gulp `gulp server`.
